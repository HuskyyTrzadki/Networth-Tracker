# Page Override - Portfolio Dashboard

Use `MASTER.md` as baseline. This file only lists dashboard-specific rules.

## Visual Priority
1. Net value hero
2. Value/performance chart
3. Allocation + currency exposure
4. Movers / transactions / dividends

## Override Rules
- Keep the first two blocks visually dominant with slightly stronger card elevation than lower sections.
- Controls (`Tryb`, `Zakres`, `Waluta`) should stay compact, high-contrast, and stable in width to avoid jumpiness.
- Chart grid and zero lines should remain visible but subdued.
- Empty/loading states should keep plain Polish guidance and avoid decorative visuals.

## Data Readability
- Use mono + tabular numbers for all key values and deltas.
- Positive/negative colors should be muted and never fluorescent.
- Keep spacing tight but breathable (`8px` rhythm inside dense cards).
